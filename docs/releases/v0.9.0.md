## modelmux v0.9.0

Real-time dispatch status tracking and live monitoring.

实时分发状态追踪和活动监控。

### Highlights / 亮点

- **Status Tracking / 状态追踪**: Active dispatches write real-time status files to `~/.config/modelmux/status/`
  运行中的分发任务实时写入状态文件到 `~/.config/modelmux/status/` 目录。

- **`modelmux status` / 状态监控**: View currently running dispatches with elapsed time and output preview
  查看当前正在运行的分发任务，包括已用时间和输出预览。

- **`modelmux status -w` / 实时刷新**: Live-refresh mode, updates every second (like `watch`)
  实时刷新模式，每秒更新一次（类似 `watch` 命令）。

- **mux_check Enhancement / mux_check 增强**: Now includes `_active_dispatches` count in output
  输出中新增 `_active_dispatches` 字段，显示当前活跃的分发数量。

### Example / 示例

```bash
$ modelmux status
modelmux — Active Dispatches
----------------------------------------
  ● abc123de  codex    12.3s  Implement REST API for user management
    └─ Creating handler functions...
```

### Install / 安装

```bash
uvx modelmux
```
