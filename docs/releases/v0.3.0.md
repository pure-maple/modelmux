## modelmux v0.3.0

Audit logging and policy engine for governance and rate limiting.

### Highlights

- **Audit Log**: All dispatches logged to `audit.jsonl` with timestamps, providers, durations, and outcomes.
- **Policy Engine**: `policy.json` configuration for rate limiting, provider blocking, and usage governance.
- **Rate Limiting**: Per-provider call frequency control to prevent abuse.

### Install

```bash
uvx modelmux
# or
pip install modelmux==0.3.0
```

## modelmux v0.3.0

新增审计日志和策略引擎，支持治理和速率限制。

### 亮点

- **审计日志**：所有调度记录写入 `audit.jsonl`，包含时间戳、提供方、耗时和执行结果。
- **策略引擎**：通过 `policy.json` 配置速率限制、提供方封禁和用量治理策略。
- **速率限制**：按提供方进行调用频率控制，防止滥用。

### 安装

```bash
uvx modelmux
# 或
pip install modelmux==0.3.0
```
