## modelmux v0.3.0

Audit logging and policy engine for governance and rate limiting.

新增审计日志和策略引擎，支持治理和速率限制。

### Highlights / 亮点

- **Audit Log / 审计日志**: All dispatches logged to `audit.jsonl` with timestamps, providers, durations, and outcomes
  所有调度记录写入 `audit.jsonl`，包含时间戳、提供方、耗时和执行结果。

- **Policy Engine / 策略引擎**: `policy.json` configuration for rate limiting, provider blocking, and usage governance
  通过 `policy.json` 配置速率限制、提供方封禁和用量治理策略。

- **Rate Limiting / 速率限制**: Per-provider call frequency control to prevent abuse
  按提供方进行调用频率控制，防止滥用。

### Install / 安装

```bash
uvx modelmux
# or
pip install modelmux==0.3.0
```
