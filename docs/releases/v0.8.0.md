## modelmux v0.8.0

CLI subcommands: `init`, `check`, `version`.

CLI 子命令架构：`init`、`check`、`version`。

### Highlights / 亮点

- **`modelmux init` / 交互式配置向导**: Interactive wizard that detects installed CLIs, generates `profiles.toml` and `policy.json`
- **`modelmux check` / 可用性检查**: Quick scan showing which model CLIs are installed and their paths
- **`modelmux version` / 版本显示**: Print current modelmux version
- **argparse Architecture / 子命令架构**: CLI refactored to extensible subcommand pattern

### Example / 示例

```bash
$ modelmux check
modelmux v0.8.0

  [+] codex    /usr/local/bin/codex
  [+] gemini   /usr/local/bin/gemini
  [+] claude   /usr/local/bin/claude
  [-] ollama   not found
```

### Install / 安装

```bash
uvx modelmux
```
