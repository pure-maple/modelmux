## modelmux v0.8.0

CLI subcommands: `init`, `check`, `version`.

### Highlights

- **`modelmux init`**: Interactive wizard that detects installed CLIs, generates `profiles.toml` and `policy.json`.
- **`modelmux check`**: Quick scan showing which model CLIs are installed and their paths.
- **`modelmux version`**: Print current modelmux version.
- **argparse Architecture**: CLI refactored to extensible subcommand pattern.

### Example

```bash
$ modelmux check
modelmux v0.8.0

  [+] codex    /usr/local/bin/codex
  [+] gemini   /usr/local/bin/gemini
  [+] claude   /usr/local/bin/claude
  [-] ollama   not found
```

### Install

```bash
uvx modelmux
```

## modelmux v0.8.0

CLI 子命令架构：`init`、`check`、`version`。

### 亮点

- **`modelmux init` 交互式配置向导**：交互式向导，自动检测已安装的 CLI 工具，生成 `profiles.toml` 和 `policy.json` 配置文件。
- **`modelmux check` 可用性检查**：快速扫描各模型 CLI 的安装状态和路径。
- **`modelmux version` 版本显示**：显示当前 modelmux 版本号。
- **子命令架构**：CLI 重构为可扩展的子命令模式。

### 示例

```bash
$ modelmux check
modelmux v0.8.0

  [+] codex    /usr/local/bin/codex
  [+] gemini   /usr/local/bin/gemini
  [+] claude   /usr/local/bin/claude
  [-] ollama   not found
```

### 安装

```bash
uvx modelmux
```
