## modelmux v0.5.0

Rebranding from `collab-hub` to `modelmux`.

### Breaking Changes

- **Package Renamed**: `collab-hub` → `modelmux`.
- **Tools Renamed**: `collab_dispatch` → `mux_dispatch`, `collab_check` → `mux_check`.
- **Migration**: Uninstall old package and reinstall: `pip uninstall collab-hub && pip install modelmux`.

### Install

```bash
uvx modelmux
# or
pip install modelmux==0.5.0
```
