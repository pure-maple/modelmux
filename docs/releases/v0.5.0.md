## modelmux v0.5.0

Rebranding from `collab-hub` to `modelmux`.

项目重命名：从 `collab-hub` 更名为 `modelmux`。

### Breaking Changes / 破坏性变更

- **Package renamed / 包重命名**: `collab-hub` → `modelmux`
- **Tool renamed / 工具重命名**: `collab_dispatch` → `mux_dispatch`, `collab_check` → `mux_check`
- **Migration / 迁移**: Uninstall old package and reinstall: `pip uninstall collab-hub && pip install modelmux`

### Install / 安装

```bash
uvx modelmux
# or
pip install modelmux==0.5.0
```
