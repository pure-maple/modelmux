## modelmux v0.14.0

Result comparison and diff analysis for broadcast results.

### Highlights

- **Result Comparison**: `mux_broadcast(compare=True)` adds automated comparison analysis.
- **Jaccard Similarity**: Quantitative similarity scoring between provider outputs.
- **Speed Ranking**: Providers ranked by response time.
- **Unique Terms**: Highlights terms each provider mentioned that others didn't.
- **Agreement Score**: Overall consensus metric across all provider responses.

### Example

```
mux_broadcast(task="Evaluate REST vs GraphQL", compare=True)
→ Results from codex, gemini, claude
→ Comparison: agreement_score=0.72, unique insights per provider
```

### Install

```bash
uvx modelmux
```

## modelmux v0.14.0

广播结果对比和 Diff 分析。

### 亮点

- **结果对比**：`mux_broadcast(compare=True)` 新增自动对比分析功能。
- **Jaccard 相似度**：基于 Jaccard 系数对各 provider 输出进行量化相似度评分。
- **速度排名**：按响应时间对 provider 进行排名。
- **独有术语**：高亮显示每个 provider 独有的、其他模型未提及的术语。
- **共识分数**：跨所有 provider 响应的总体共识度指标。

### 示例

```
mux_broadcast(task="Evaluate REST vs GraphQL", compare=True)
→ codex、gemini、claude 的结果
→ 对比：agreement_score=0.72，每个 provider 的独特见解
```

### 安装

```bash
uvx modelmux
```
