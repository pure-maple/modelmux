## modelmux v0.14.0

Result comparison and diff analysis for broadcast results.

广播结果对比和 Diff 分析。

### Highlights / 亮点

- **Result Comparison / 结果对比**: `mux_broadcast(compare=True)` adds automated comparison analysis
- **Jaccard Similarity / Jaccard 相似度**: Quantitative similarity scoring between provider outputs
- **Speed Ranking / 速度排名**: Providers ranked by response time
- **Unique Terms / 独有术语**: Highlights terms each provider mentioned that others didn't
- **Agreement Score / 共识分数**: Overall consensus metric across all provider responses

### Example / 示例

```
mux_broadcast(task="Evaluate REST vs GraphQL", compare=True)
→ Results from codex, gemini, claude
→ Comparison: agreement_score=0.72, unique insights per provider
```

### Install / 安装

```bash
uvx modelmux
```
