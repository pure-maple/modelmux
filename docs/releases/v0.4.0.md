## modelmux v0.4.0

CI/CD automation and registry integration.

### Highlights

- **Trusted Publisher**: GitHub Actions auto-publish to PyPI on tag push via OIDC (no API tokens needed).
- **MCP Registry**: Listed as `io.github.pure-maple/modelmux` for discovery by MCP clients.
- **CI Pipeline**: Automated testing and publishing on every release tag.

### Install

```bash
# Claude Code one-liner
claude mcp add modelmux -s user -- uvx modelmux
# or
pip install modelmux==0.4.0
```

## modelmux v0.4.0

CI/CD 自动化和注册表集成。

### 亮点

- **可信发布**：通过 GitHub Actions + OIDC 实现标签推送时自动发布到 PyPI，无需 API 令牌。

- **CI 流水线**：每次发布标签自动触发测试和发布流程。

### 安装

```bash

claude mcp add modelmux -s user -- uvx modelmux
```
