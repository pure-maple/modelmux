## modelmux v0.4.0

CI/CD automation and registry integration.

CI/CD 自动化和注册表集成。

### Highlights / 亮点

- **Trusted Publisher / 可信发布**: GitHub Actions auto-publish to PyPI on tag push via OIDC (no API tokens needed)
- **MCP Registry / MCP 注册表**: Listed as `io.github.pure-maple/modelmux` for discovery by MCP clients
- **CI Pipeline / CI 流水线**: Automated testing and publishing on every release tag

### Install / 安装

```bash
# Claude Code one-liner
claude mcp add modelmux -s user -- uvx modelmux
# or
pip install modelmux==0.4.0
```
