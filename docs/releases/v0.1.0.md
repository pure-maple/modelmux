## modelmux v0.1.0

Initial prototype — multi-model AI collaboration MCP server.

首个原型版本 — 多模型 AI 协作 MCP 服务器。

### Highlights / 亮点

- **3 CLI Adapters / 三大 CLI 适配器**: Codex (JSONL output), Gemini (stream-json), Claude (plain text)
- **Smart Routing / 智能路由**: `provider="auto"` keyword-based task routing
- **Session Continuity / 会话连续性**: Multi-turn conversations via `session_id`
- **MCP Server / MCP 服务器**: FastMCP-based stdio transport, works with any MCP client

### Supported Platforms / 支持平台

| CLI | Command | Output Format |
|-----|---------|---------------|
| Codex | `codex exec --json` | JSONL structured |
| Gemini | `gemini -p -o stream-json` | Stream JSON |
| Claude | `claude -p` | Plain text |

### Install / 安装

```bash
uvx modelmux
# or
pip install modelmux==0.1.0
```
