## modelmux v0.1.0

Initial prototype — multi-model AI collaboration MCP server.

### Highlights

- **3 CLI Adapters**: Codex (JSONL structured output), Gemini (stream JSON), Claude (plain text).
- **Smart Routing**: `provider="auto"` keyword-based task routing.
- **Session Continuity**: Multi-turn conversations via `session_id`.
- **MCP Server**: FastMCP-based stdio transport, works with any MCP client.

### Supported Platforms

| CLI | Command | Output Format |
|-----|---------|---------------|
| Codex | `codex exec --json` | JSONL structured |
| Gemini | `gemini -p -o stream-json` | Stream JSON |
| Claude | `claude -p` | Plain text |

### Install

```bash
uvx modelmux
# or
pip install modelmux==0.1.0
```

## modelmux v0.1.0

首个原型版本 — 多模型 AI 协作 MCP 服务器。

### 亮点

- **三大 CLI 适配器**：支持 Codex（JSONL 结构化输出）、Gemini（流式 JSON）、Claude（纯文本）三种 CLI 适配器。
- **智能路由**：通过 `provider="auto"` 实现基于关键词的智能任务路由。
- **会话连续性**：通过 `session_id` 支持多轮对话，保持上下文连续性。
- **MCP 服务器**：基于 FastMCP 的 stdio 传输协议，兼容所有 MCP 客户端。

### 支持平台

| CLI | 命令 | 输出格式 |
|-----|------|----------|
| Codex | `codex exec --json` | JSONL 结构化 |
| Gemini | `gemini -p -o stream-json` | 流式 JSON |
| Claude | `claude -p` | 纯文本 |

### 安装

```bash
uvx modelmux
# 或
pip install modelmux==0.1.0
```
