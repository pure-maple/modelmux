## modelmux v0.6.0

Automatic failover and MCP progress notifications.

### Highlights

- **Failover**: When a provider fails (timeout, crash, error), automatically retries with the next available provider.
- **MCP Progress**: Real-time progress updates via `ctx.info()` during long-running dispatches.
- **Failover Tracking**: `failover_from` field in results identifies the original failed provider.

### Example

```
mux_dispatch(provider="codex", task="...")
# If codex fails → auto-retries with gemini → then claude
# Result includes: failover_from="codex"
```

### Install

```bash
uvx modelmux
# or
pip install modelmux==0.6.0
```

## modelmux v0.6.0

自动容错切换和 MCP 进度通知。

### 亮点

- **自动容错**：当提供方出现故障（超时、崩溃、报错）时，自动切换到下一个可用的提供方重试。
- **MCP 进度通知**：在长时间运行的调度过程中，通过 `ctx.info()` 发送实时进度更新。
- **容错追踪**：结果中的 `failover_from` 字段标识最初失败的提供方，便于追踪容错路径。

### 示例

```
mux_dispatch(provider="codex", task="...")
# 若 codex 失败 → 自动切换到 gemini 重试 → 再失败则切换到 claude
# 结果包含: failover_from="codex"
```

### 安装

```bash
uvx modelmux
# 或
pip install modelmux==0.6.0
```
