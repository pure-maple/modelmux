## modelmux v0.6.0

Automatic failover and MCP progress notifications.

自动容错切换和 MCP 进度通知。

### Highlights / 亮点

- **Failover / 自动容错**: When a provider fails (timeout, crash, error), automatically retries with the next available provider
- **MCP Progress / MCP 进度通知**: Real-time progress updates via `ctx.info()` during long-running dispatches
- **Failover Tracking / 容错追踪**: `failover_from` field in results identifies the original failed provider

### Example / 示例

```
mux_dispatch(provider="codex", task="...")
# If codex fails → auto-retries with gemini → then claude
# Result includes: failover_from="codex"
```

### Install / 安装

```bash
uvx modelmux
# or
pip install modelmux==0.6.0
```
