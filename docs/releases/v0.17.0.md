## modelmux v0.17.0

Smart Routing v2 — adaptive routing powered by history data.

### Highlights

- **History-Aware Routing**: `provider="auto"` now uses past dispatch data to make smarter routing decisions.
- **Weighted Scoring**: Keyword match (60%) + historical performance (40%) combined scoring.
- **Performance Metrics**: History scoring considers success rate (70%) and response speed (30%).
- **Adaptive Weights**: History weight scales with data volume — 40% at >=5 calls, 25% at 2-4, keyword-only below 2.
- **Standalone Module**: Routing logic extracted to `routing.py` with 13 dedicated tests.

### How It Works

```
Task: "implement REST API"
  +-- Keyword score: codex=0.8, gemini=0.5, claude=0.6
  +-- History score: codex=0.9 (95% success, fast), gemini=0.7
  +-- Final: codex=0.84 -> selected
```

### Install

```bash
uvx modelmux
```

## modelmux v0.17.0

智能路由 v2 — 基于历史数据的自适应路由。

### 亮点

- **历史感知路由**：`provider="auto"` 现在利用历史调度数据做出更智能的路由决策。
- **加权评分**：关键词匹配（60%）+ 历史表现（40%）组合评分。
- **性能指标**：历史评分综合考量成功率（70%）和响应速度（30%）。
- **自适应权重**：历史权重随数据量自适应调整——调用 >=5 次时占 40%，2-4 次占 25%，不足 2 次时仅用关键词评分。
- **独立模块**：路由逻辑抽取为独立模块 `routing.py`，配有 13 个专项测试。

### 工作原理

```
Task: "implement REST API"
  +-- Keyword score: codex=0.8, gemini=0.5, claude=0.6
  +-- History score: codex=0.9 (95% success, fast), gemini=0.7
  +-- Final: codex=0.84 -> selected
```

### 安装

```bash
uvx modelmux
```
