## modelmux v0.17.0

Smart Routing v2 — adaptive routing powered by history data.

智能路由 v2 — 基于历史数据的自适应路由。

### Highlights / 亮点

- **History-Aware Routing / 历史感知路由**: `provider="auto"` now uses past dispatch data to make smarter routing decisions
- **Weighted Scoring / 加权评分**: Keyword match (60%) + historical performance (40%) combined scoring
- **Performance Metrics / 性能指标**: History scoring considers success rate (70%) and response speed (30%)
- **Adaptive Weights / 自适应权重**: History weight scales with data volume — 40% at ≥5 calls, 25% at 2-4, keyword-only below 2
- **Standalone Module / 独立模块**: Routing logic extracted to `routing.py` with 13 dedicated tests

### How It Works / 工作原理

```
Task: "implement REST API"
  ├─ Keyword score: codex=0.8, gemini=0.5, claude=0.6
  ├─ History score: codex=0.9 (95% success, fast), gemini=0.7
  └─ Final: codex=0.84 → selected ✓
```

### Install / 安装

```bash
uvx modelmux
```
