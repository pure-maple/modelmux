## modelmux v0.10.0

TUI configuration panel powered by Textual.

基于 Textual 的 TUI 配置面板。

### Highlights / 亮点

- **`modelmux config` / TUI 配置面板**: Full terminal UI for editing routing rules, policies, and profiles
  完整的终端 UI 界面，可编辑路由规则、策略和配置文件。

- **Three Tabs / 三个标签页**: Overview (adapters & status), Routing (rules editor), Policy (rate limits & blocks)
  概览（适配器与状态）、路由（规则编辑器）、策略（速率限制与黑名单）。

- **Optional Dependency / 可选依赖**: `pip install modelmux[tui]` — TUI only loaded when needed
  `pip install modelmux[tui]` 安装 TUI 支持，仅在使用时加载。

- **Live Editing / 实时编辑**: Changes saved directly to `profiles.toml` and `policy.json`
  修改实时保存到 `profiles.toml` 和 `policy.json`。

### Example / 示例

```bash
# Install with TUI support
pip install modelmux[tui]
# Or run directly
uvx --with textual modelmux config
```

### Install / 安装

```bash
uvx modelmux
# For TUI:
pip install modelmux[tui]
```
