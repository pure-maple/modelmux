## modelmux v0.10.0

TUI configuration panel powered by Textual.

### Highlights

- **`modelmux config`**: Full terminal UI for editing routing rules, policies, and profiles.
- **Three Tabs**: Overview (adapters & status), Routing (rules editor), Policy (rate limits & blocks).
- **Optional Dependency**: `pip install modelmux[tui]` -- TUI only loaded when needed.
- **Live Editing**: Changes saved directly to `profiles.toml` and `policy.json`.

### Example

```bash
# Install with TUI support
pip install modelmux[tui]
# Or run directly
uvx --with textual modelmux config
```

### Install

```bash
uvx modelmux
# For TUI:
pip install modelmux[tui]
```

## modelmux v0.10.0

基于 Textual 的 TUI 配置面板。

### 亮点

- **`modelmux config` TUI 配置面板**：完整的终端 UI 界面，可编辑路由规则、策略和配置文件。
- **三个标签页**：概览（适配器与状态）、路由（规则编辑器）、策略（速率限制与黑名单）。
- **可选依赖**：`pip install modelmux[tui]` 安装 TUI 支持，仅在使用时加载。
- **实时编辑**：修改实时保存到 `profiles.toml` 和 `policy.json`。

### 示例

```bash
# 安装 TUI 支持
pip install modelmux[tui]
# 或直接运行
uvx --with textual modelmux config
```

### 安装

```bash
uvx modelmux
# TUI 支持：
pip install modelmux[tui]
```
