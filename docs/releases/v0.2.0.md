## modelmux v0.2.0

First PyPI release with caller detection and profile system.

### Highlights

- **Caller Detection**: Auto-detects calling platform via MCP `clientInfo`, excludes caller from routing to prevent self-dispatch loops.
- **Profile System**: Named profiles (`budget`, `china`, etc.) for customizing model preferences and routing rules.
- **Custom Routing Rules**: TOML/JSON/YAML config with keyword and regex matching.
- **PyPI Release**: `pip install modelmux` now available.

### Install

```bash
uvx modelmux
# or
pip install modelmux==0.2.0
```

## modelmux v0.2.0

首次 PyPI 发布，新增调用方检测和 Profile 配置系统。

### 亮点

- **调用方检测**：通过 MCP `clientInfo` 自动检测调用方平台，将其从路由中排除以避免自我调度循环。
- **配置文件系统**：支持命名配置（如 `budget`、`china` 等），用于自定义模型偏好和路由规则。
- **自定义路由规则**：通过 TOML/JSON/YAML 配置文件，支持关键词和正则表达式匹配的路由规则。
- **发布到 PyPI**：现已支持 `pip install modelmux` 安装。

### 安装

```bash
uvx modelmux
# 或
pip install modelmux==0.2.0
```
