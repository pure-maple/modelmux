## modelmux v0.2.0

First PyPI release with caller detection and profile system.

首次 PyPI 发布，新增调用方检测和 Profile 配置系统。

### Highlights / 亮点

- **Caller Detection / 调用方检测**: Auto-detects calling platform via MCP `clientInfo`, excludes caller from routing to prevent self-dispatch loops
  通过 MCP `clientInfo` 自动检测调用方平台，将其从路由中排除以避免自我调度循环。

- **Profile System / 配置文件系统**: Named profiles (`budget`, `china`, etc.) for customizing model preferences and routing rules
  支持命名配置（如 `budget`、`china` 等），用于自定义模型偏好和路由规则。

- **Custom Routing Rules / 自定义路由**: TOML/JSON/YAML config with keyword/regex matching
  通过 TOML/JSON/YAML 配置文件，支持关键词和正则表达式匹配的路由规则。

- **PyPI Release / 发布到 PyPI**: `pip install modelmux` now available
  现已支持 `pip install modelmux` 安装。

### Install / 安装

```bash
uvx modelmux
# or
pip install modelmux==0.2.0
```
