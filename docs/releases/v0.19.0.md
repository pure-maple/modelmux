## modelmux v0.19.0

A2A HTTP Server — full Agent-to-Agent protocol transport layer.

### Highlights

- **A2A HTTP Server**: Starlette-based JSON-RPC 2.0 server implementing the A2A protocol v0.3.0.
- **Agent Card**: `GET /.well-known/agent.json` — auto-generated capability discovery endpoint.
- **4 JSON-RPC Methods**:
  - `tasks/send` — Synchronous task execution.
  - `tasks/get` — Query task state and results.
  - `tasks/cancel` — Cancel running tasks.
  - `tasks/sendSubscribe` — SSE streaming with real-time progress.
- **TaskStore Persistence**: Completed tasks persisted to `~/.config/modelmux/a2a-tasks.jsonl`, restored on restart.
- **Authentication**: Bearer token auth with constant-time comparison (`--token` or `MODELMUX_A2A_TOKEN` env var).
- **CORS + Health Check**: Full CORS support and `/health` endpoint for load balancers.
- **Codex UTF-8 Workaround**: Non-ASCII workdir auto-creates temporary ASCII symlink with PWD env override.

### Usage

```bash
# Start the A2A server
modelmux a2a-server --port 41520

# With authentication
modelmux a2a-server --token my-secret

# In-memory only (no persistence)
modelmux a2a-server --no-persist
```

### A2A Client Example

```python
import httpx

resp = httpx.post("http://localhost:41520/", json={
    "jsonrpc": "2.0", "id": 1,
    "method": "tasks/send",
    "params": {
        "message": {
            "role": "user",
            "parts": [{"type": "text", "text": "Review this code"}]
        },
        "metadata": {"pattern": "review"}
    }
})
result = resp.json()["result"]
```

## modelmux v0.19.0

A2A HTTP 服务器 — 完整的 Agent-to-Agent 协议传输层。

### 亮点

- **A2A HTTP 服务器**：基于 Starlette 的 JSON-RPC 2.0 服务器，实现 A2A 协议 v0.3.0。
- **Agent 能力卡片**：`GET /.well-known/agent.json` — 自动生成的能力发现端点。
- **4 个 JSON-RPC 方法**：
  - `tasks/send` — 同步任务执行。
  - `tasks/get` — 查询任务状态和结果。
  - `tasks/cancel` — 取消正在运行的任务。
  - `tasks/sendSubscribe` — SSE 流式传输，支持实时进度更新。
- **任务持久化**：已完成的任务持久化到 `~/.config/modelmux/a2a-tasks.jsonl`，重启时自动恢复。
- **认证**：Bearer token 认证，使用常量时间比较（通过 `--token` 参数或 `MODELMUX_A2A_TOKEN` 环境变量配置）。
- **跨域 + 健康检查**：完整的 CORS 跨域支持和 `/health` 健康检查端点，可用于负载均衡器。
- **Codex UTF-8 修复**：非 ASCII 工作目录自动创建临时 ASCII 符号链接并覆盖 PWD 环境变量。

### 使用

```bash

# 启用认证
modelmux a2a-server --token my-secret

# 纯内存模式（不持久化）
```
