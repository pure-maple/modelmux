## modelmux v0.19.0

A2A HTTP Server — full Agent-to-Agent protocol transport layer.

A2A HTTP 服务器 — 完整的 Agent-to-Agent 协议传输层。

### Highlights / 亮点

- **A2A HTTP Server / A2A HTTP 服务器**: Starlette-based JSON-RPC 2.0 server implementing the A2A protocol v0.3.0
- **Agent Card / Agent 能力卡片**: `GET /.well-known/agent.json` — auto-generated capability discovery endpoint
- **4 JSON-RPC Methods / 4 个 JSON-RPC 方法**:
  - `tasks/send` — Synchronous task execution
  - `tasks/get` — Query task state and results
  - `tasks/cancel` — Cancel running tasks
  - `tasks/sendSubscribe` — SSE streaming with real-time progress
- **TaskStore Persistence / 任务持久化**: Completed tasks persisted to `~/.config/modelmux/a2a-tasks.jsonl`, restored on restart
- **Authentication / 认证**: Bearer token auth with constant-time comparison (`--token` or `MODELMUX_A2A_TOKEN` env var)
- **CORS + Health Check / 跨域 + 健康检查**: Full CORS support and `/health` endpoint for load balancers
- **Codex UTF-8 Workaround / Codex UTF-8 修复**: Non-ASCII workdir auto-creates temporary ASCII symlink with PWD env override

### Usage / 使用

```bash
# Start the A2A server
modelmux a2a-server --port 41520

# With authentication
modelmux a2a-server --token my-secret

# In-memory only (no persistence)
modelmux a2a-server --no-persist
```

### A2A Client Example / A2A 客户端示例

```python
import httpx

resp = httpx.post("http://localhost:41520/", json={
    "jsonrpc": "2.0", "id": 1,
    "method": "tasks/send",
    "params": {
        "message": {
            "role": "user",
            "parts": [{"type": "text", "text": "Review this code"}]
        },
        "metadata": {"pattern": "review"}
    }
})
result = resp.json()["result"]
```
