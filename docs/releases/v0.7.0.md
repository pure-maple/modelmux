## modelmux v0.7.0

Ollama adapter for local models.

### Highlights

- **Ollama Adapter**: Run local models (DeepSeek, Llama, Qwen, Mistral, etc.) through modelmux.
- **Model Selection**: `mux_dispatch(provider="ollama", model="deepseek-r1")` to specify any Ollama model.
- **Output Filtering**: Automatically filters download progress bars from Ollama output.

### Supported Local Models

| Model | Command |
|-------|---------|
| DeepSeek R1 | `model="deepseek-r1"` |
| Llama 3 | `model="llama3"` |
| Qwen 2 | `model="qwen2"` |
| Mistral | `model="mistral"` |

### Install

```bash
# Requires Ollama installed: https://ollama.com
uvx modelmux
# or
pip install modelmux==0.7.0
```

## modelmux v0.7.0

新增 Ollama 适配器，支持本地模型。

### 亮点

- **Ollama 适配器**：通过 modelmux 运行本地模型（DeepSeek、Llama、Qwen、Mistral 等）。
- **模型选择**：使用 `mux_dispatch(provider="ollama", model="deepseek-r1")` 指定任意 Ollama 模型。
- **输出过滤**：自动过滤 Ollama 输出中的下载进度条，保持输出整洁。

### 支持的本地模型

| 模型 | 命令 |
|------|------|
| DeepSeek R1 | `model="deepseek-r1"` |
| Llama 3 | `model="llama3"` |
| Qwen 2 | `model="qwen2"` |
| Mistral | `model="mistral"` |

### 安装

```bash
# 需要先安装 Ollama: https://ollama.com
uvx modelmux
# 或
pip install modelmux==0.7.0
```
