## modelmux v0.7.0

Ollama adapter for local models.

新增 Ollama 适配器，支持本地模型。

### Highlights / 亮点

- **Ollama Adapter / Ollama 适配器**: Run local models (DeepSeek, Llama, Qwen, Mistral, etc.) through modelmux
- **Model Selection / 模型选择**: `mux_dispatch(provider="ollama", model="deepseek-r1")` to specify any Ollama model
- **Output Filtering / 输出过滤**: Automatically filters download progress bars from Ollama output

### Supported Local Models / 支持的本地模型

| Model | Command |
|-------|---------|
| DeepSeek R1 | `model="deepseek-r1"` |
| Llama 3 | `model="llama3"` |
| Qwen 2 | `model="qwen2"` |
| Mistral | `model="mistral"` |

### Install / 安装

```bash
# Requires Ollama installed: https://ollama.com
uvx modelmux
```
