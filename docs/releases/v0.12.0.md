## modelmux v0.12.0

Dispatch history — persistent storage and analytics.

分发历史 — 持久化存储和统计分析。

### Highlights / 亮点

- **`mux_history` / 历史查询工具**: New MCP tool to query past dispatches by provider, status, or time range
  新增 MCP 工具，按 provider、状态或时间范围查询历史分发记录。

- **Persistent Storage / 持久化存储**: All dispatch results saved to `history.jsonl` with automatic rotation (10MB cap)
  所有分发结果持久化到 `history.jsonl`，自动轮转（上限 10MB）。

- **`modelmux history` / CLI 历史查看**: View recent dispatch records from the command line
  通过命令行查看近期的分发记录。

- **`modelmux history --stats` / 统计分析**: Per-provider statistics including success rate and average latency
  按 provider 统计成功率和平均延迟等指标。

### Example / 示例

```bash
$ modelmux history --stats
modelmux — History Statistics
--------------------------------------------------
  Total dispatches: 42
  codex:   18 calls  94.4% success  avg 8.2s
  gemini:  15 calls  86.7% success  avg 5.1s
  claude:   9 calls  100.0% success  avg 12.3s
```

### Install / 安装

```bash
uvx modelmux
```
