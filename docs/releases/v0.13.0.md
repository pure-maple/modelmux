## modelmux v0.13.0

Workflow templates — multi-step pipeline orchestration.

工作流模板 — 多步流水线编排。

### Highlights / 亮点

- **`mux_workflow` / 工作流工具**: New MCP tool for multi-step pipeline execution across providers
- **Built-in Workflows / 内置工作流**: `review` (codex implements → claude reviews) and `consensus` (multi-model synthesis)
- **Custom Workflows / 自定义工作流**: Define your own pipelines in `profiles.toml` under `[workflows.*]`
- **Output Chaining / 输出链式传递**: Steps reference previous outputs via `{step_name}` placeholders

### Example / 示例

```
mux_workflow(workflow="review", task="Implement user auth module")
→ Step 1: codex implements the code
→ Step 2: claude reviews the implementation
→ Final: combined result with implementation + review feedback
```

### Install / 安装

```bash
uvx modelmux
# or
pip install modelmux==0.13.0
```
