## modelmux v0.16.0

Streaming output — real-time progress for all MCP tools.

流式输出 — 所有 MCP 工具支持实时进度更新。

### Highlights / 亮点

- **Streaming Progress / 流式进度**: All MCP tools (`mux_dispatch`, `mux_broadcast`, `mux_workflow`) now emit line-by-line progress updates.
  所有 MCP 工具（`mux_dispatch`、`mux_broadcast`、`mux_workflow`）现在支持逐行输出进度更新。

- **on_progress Callback / 进度回调**: Each output line triggers a progress notification to the MCP client.
  每行输出都会触发一次进度通知发送到 MCP 客户端。

- **Throttled I/O / 节流写入**: Status file updates throttled to 0.5s intervals to prevent excessive disk I/O.
  状态文件写入限流为 0.5 秒间隔，避免过度磁盘 I/O。

- **Output Tracking / 输出追踪**: `DispatchStatus.output_lines` field tracks total output line count.
  `DispatchStatus.output_lines` 字段追踪输出总行数。

### How It Works / 工作原理

```
MCP Client ← on_progress("Line 1: Setting up...") ← modelmux ← codex
MCP Client ← on_progress("Line 2: Running tests...") ← modelmux ← codex
MCP Client ← on_progress("Line 3: All tests passed") ← modelmux ← codex
```

### Install / 安装

```bash
uvx modelmux
```
