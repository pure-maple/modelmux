## modelmux v0.11.0

Parallel broadcast — send the same task to multiple models simultaneously.

并行广播 — 将同一任务同时发送给多个模型。

### Highlights / 亮点

- **`mux_broadcast` / 并行广播工具**: New MCP tool to dispatch the same task to all available providers in parallel
- **Async Parallel / 异步并行**: Uses `asyncio.gather` for true concurrent execution — significantly faster than sequential calls
- **Auto Provider Selection / 自动选择 Provider**: Automatically selects all available providers (excluding the caller)
- **Aggregated Results / 聚合结果**: Returns results from all providers with success/error statistics

### Example / 示例

```
mux_broadcast(task="Review this API design for security issues")
→ Codex, Gemini, Claude all respond in parallel
→ Compare perspectives from 3 different models
```

### Install / 安装

```bash
uvx modelmux
```
