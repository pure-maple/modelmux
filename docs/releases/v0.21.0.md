## modelmux v0.21.0

Cost tracking, documentation overhaul, and repository cleanup.

### Highlights

- **Cost Tracking**: Automatic token usage extraction from Codex (`turn.completed`) and Gemini (`usageMetadata`) responses. New `costs.py` module with per-model pricing and aggregation. Query via `mux_history(costs=True)` or `modelmux history --stats --costs`.
- **Documentation Overhaul**: Complete rewrite of both English and Chinese READMEs — all 6 MCP tools, A2A protocol, cost tracking, full CLI commands, updated architecture diagram, and feature summary table.
- **Repository Cleanup**: Removed 1,900+ lines of content not suitable for a public repository — raw AI consultation session logs, internal development notes, and code review process records. Replaced with a clean architecture consultation summary.
- **Security Hardening**: Added `.mcpregistry_*`, `.env`, `*.key`, `*.pem` to `.gitignore` to prevent accidental credential commits.

### Test Coverage

183 tests passing (22 new cost tracking tests)

### Install

```bash
uvx modelmux
# or
pip install modelmux==0.21.0
```
