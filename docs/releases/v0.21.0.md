## modelmux v0.21.0

Cost tracking, documentation overhaul, and repository cleanup.

成本追踪、文档全面更新与仓库清理。

### Highlights / 亮点

- **Cost Tracking / 成本追踪**: Automatic token usage extraction from Codex (`turn.completed`) and Gemini (`usageMetadata`) responses. New `costs.py` module with per-model pricing and aggregation. Query via `mux_history(costs=True)` or `modelmux history --stats --costs`.
  自动从 Codex 和 Gemini 响应中提取 token 用量。新增 `costs.py` 模块，包含多模型定价表与聚合统计。通过 MCP 工具或 CLI 查询成本。

- **Documentation Overhaul / 文档全面更新**: Complete rewrite of both English and Chinese READMEs — all 6 MCP tools, A2A protocol, cost tracking, full CLI commands, updated architecture diagram, and feature summary table.
  中英文 README 完全重写，涵盖全部 6 个 MCP 工具、A2A 协议、成本追踪、完整 CLI 命令、架构图和功能一览表。

- **Repository Cleanup / 仓库清理**: Removed 1,900+ lines of content not suitable for a public repository — raw AI consultation session logs, internal development notes, and code review process records. Replaced with a clean architecture consultation summary.
  移除 1,900+ 行不适合公开仓库的内容（原始会话日志、内部开发笔记、代码审查记录），替换为精简版架构咨询摘要。

- **Security Hardening / 安全加固**: Added `.mcpregistry_*`, `.env`, `*.key`, `*.pem` to `.gitignore` to prevent accidental credential commits.
  在 `.gitignore` 中添加凭证文件模式，防止意外提交敏感信息。

183 个测试通过（新增 22 个成本追踪测试）
