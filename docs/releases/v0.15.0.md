## modelmux v0.15.0

Custom provider plugins — bring any CLI tool as a provider.

自定义 Provider 插件 — 将任意 CLI 工具注册为 Provider。

### Highlights / 亮点

- **Custom Providers / 自定义 Provider**: Register any CLI tool as a modelmux provider via `profiles.toml`.
  通过 `profiles.toml` 将任意 CLI 工具注册为 modelmux Provider。

- **GenericAdapter / 通用适配器**: Template-based command execution with `{task}` and `{workdir}` placeholders.
  基于模板的命令执行，支持 `{task}` 和 `{workdir}` 占位符。

- **Unlimited Providers / 不限 Provider**: `mux_dispatch` now accepts any registered provider name, not just the built-in 4.
  `mux_dispatch` 现在接受任意已注册的 Provider 名称，不再局限于内置的 4 个。

- **mux_check Enhancement / mux_check 增强**: Custom providers shown with `custom: true` flag.
  自定义 Provider 会以 `custom: true` 标记显示。

### Configuration / 配置

```toml
# ~/.config/modelmux/profiles.toml
[providers.mybot]
command = "mybot"
args = ["--prompt", "{task}", "--dir", "{workdir}"]
```

```
mux_dispatch(provider="mybot", task="do something")
```

### Install / 安装

```bash
uvx modelmux
```
